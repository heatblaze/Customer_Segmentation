# 🧠 Retail Customer Segmentation: Neural Overhaul

[![Market Status](https://img.shields.io/badge/Status-Production--Ready-success?style=for-the-badge&logo=rocket)](https://github.com/heatblaze/Customer_Segmentation)
[![Stack](https://img.shields.io/badge/Stack-Next.js%20%7C%20FastAPI%20%7C%20ECharts-blueviolet?style=for-the-badge)](https://github.com/heatblaze/Customer_Segmentation)

A premium, glassmorphic analytics ecosystem designed to transform raw retail data into actionable behavioral intelligence. This platform leverages dual clustering engines (HDBSCAN & GMM) and a high-performance interactive UI to provide deep insights into customer lifecycle and value segments.

---

## ✨ Key Features

### 🌪️ Neural Architecture
- **Living Mesh UI**: A dynamic, glassmorphic design system with animated gradients and interactive background nodes.
- **Dual Engine Clustering**: Toggle between standard K-Means, HDBSCAN (Density-based), and GMM (Probabilistic) for robust segmentation.
- **Zero-Clip Grid**: Optimized 12-column layout ensuring pixel-perfect visibility at 100% zoom.

### 🕵️ Behavioral Intelligence
- **Persona Shelf**: Human-readable segment profiles (e.g., *High-Value Catalysts*, *Rising Stars*) with AI-driven behavioral descriptions.
- **Fingerprint DNA**: Radar chart mapping of segmentation axes (Monetary, Loyalty, Frequency, etc.).
- **Complexity Projection**: Latent space visualization showing inter-cluster relationships.

### 📈 Intelligence Repository
- **Metric Propensity**: Comparative distribution analysis across all RFM (Recency, Frequency, Monetary) metrics.
- **Retailer Leaderboard**: High-performance ranking system with individual retailer behavioral profiling.
- **Actionable Insights**: Quantified revenue contribution and strategic recommendations.

## 🚀 Getting Started

### Prerequisites
- Node.js (v20+)
- Python (v3.10+)

### 1. Backend Setup
```bash
# Install dependencies
pip install -r requirements.txt

# Launch Analytical API
python api.py
```

### 2. Frontend Setup
```bash
cd frontend

# Install dependencies
npm install

# Start Dashboard
npm run dev
```

The application will be live at `http://localhost:3000`.
